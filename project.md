# Зоя Байкова — Квиз «Почему вес не уходит»

Бесплатный квиз-лендинг для нутрициолога Зои Байковой. Пользователь отвечает на 12 вопросов о теле, питании и самочувствии, оставляет контакт — Зоя пишет лично с персональным разбором в течение 24 часов.

## Целевая аудитория
Женщины, которые не могут похудеть, страдают от отёков, усталости, гормональных проблем.

## Файлы

| Файл | Назначение |
|------|-----------|
| `zoya_kviz.html` | Весь проект — HTML + CSS + JS в одном файле |

## Технический стек
- Чистый HTML/CSS/JS — без фреймворков
- Шрифты: Playfair Display + Nunito (Google Fonts)
- Фото Зои встроено в base64 прямо в HTML

## Как запустить
Открыть `zoya_kviz.html` в браузере — всё работает локально.

## Известные проблемы
- `submit()` не отправляет данные (только `console.log`) — TODO
- Отсутствует `id="intro-screen"` и `id="rpanel"` — баги в JS

## Структура квиза
1. Экран старта
2. Q1–Q8: вопросы про тело, энергию, отёки, питание, сон, диагнозы, цели
3. Q9–Q11: рост/вес, возраст, имя
4. Q12: контакт (Telegram/WhatsApp + телефон)
5. Экран успеха
